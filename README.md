## 개요

**노닥노닥**은 선택지를 공유하고 의견을 나누며, 고민을 해결하고 비슷한 관심사를 가진 이용자들과 소통하는 **투표형 커뮤니티** 서비스입니다.

## 실행 방법

```
git clone https://github.com/prgrms-fe-devcourse/FEDC5_NodakNodak_Noah.git
```

```
npm install
```

```
npm run dev
npm run build && npm run preview
```
